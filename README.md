# Travel Book

## Requirements 

* Docker
* Docker Compose
* Docker Repository: iamluc/host-manager

## Installation

Use this command:

```bash
cp .env.dist .env
```

This file will be used for set environment variables.


## Start application

Use this command:

```bash
make start
```

## Access to application

You can go to: [http://web.travelbook/](http://web.travelbook/)
